隐私政策

最后更新日期：2026年3月8日

1. 简介

本隐私政策适用于"个人健康数据分析平台"（以下简称"本应用"）。本应用由徐天华开发，仅供个人学习研究使用。

2. 数据收集范围

本应用通过华为Health Kit API，在获得用户明确授权后，读取以下健康数据：

  • 运动步数
  • 心率数据
  • 睡眠数据（深睡/浅睡/REM）
  • 血氧饱和度
  • 体重及体脂率
  • 运动记录

3. 数据使用方式

上述健康数据仅用于以下目的：

  • 生成个人健康趋势图表（步数、心率、睡眠等可视化）
  • 生成个人健康分析报告，供用户本人查看

4. 数据存储与安全

  • 所有健康数据仅在用户本地设备处理，不上传至任何第三方服务器
  • 本应用不存储、不缓存任何用户健康数据
  • 数据处理完全在本地完成，网络请求仅用于调用华为Health Kit API

5. 数据共享

本应用不会将您的健康数据：

  • 出售给任何第三方
  • 用于商业目的
  • 与任何其他用户或机构共享

6. 用户权利

您随时可以通过华为健康App撤销对本应用的数据访问授权。撤销授权后，本应用将无法继续访问您的健康数据。
